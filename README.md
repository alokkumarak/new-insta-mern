## Clone Repo

```sh
https://github.com/alokkumarak/new-insta-mern.git
```
## Install packages

```sh
cd client
```
```sh
npm install
```
```sh
cd ..
cd server
```
```sh
npm install
```
## run server
```sh
node server.js
```
## run client
```sh
npm start
```
